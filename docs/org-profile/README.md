**getbms** is an open source organization building free, local-first business management tools. No subscriptions, no lock-in, no cloud required.

<br/>

## What we build

| Project | Description | Stack |
|---------|-------------|-------|
| [bms](https://github.com/getbms/bms) | Business Management System - sales, inventory, finance, and contacts in one app | Flutter, Drift, Riverpod |

<br/>

## Why open source

Small businesses deserve real software. Most tools are either too expensive, too complex, or tied to a cloud subscription. We build simple, fast, offline-capable apps that anyone can run, inspect, and improve.

Everything here is:
- **Free to use** - no trial, no paywall, no feature limits
- **Open to inspect** - read every line, audit every decision
- **Yours to modify** - fork it, extend it, ship your own version
- **Local by default** - your data stays on your device
